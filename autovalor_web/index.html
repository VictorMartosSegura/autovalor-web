Cómo funciona

Tres pasos para publicar un anuncio de calidad.

01

Sube las fotos

La app guía al usuario con fotos recomendadas: frontal, posterior, lateral, interior, salpicadero y motor.

02

La IA analiza

OpenAI analiza las imágenes y devuelve marca, modelo, título, descripción, confianza y avisos.

03

Revisas y publicas

La propuesta siempre es editable. Ningún anuncio se publica automáticamente.

Funcionalidades

Todo el flujo de compraventa en una sola app.

✦ AutoValor AI

Envía fotos al backend, construye un prompt estructurado y llama a OpenAI. La respuesta incluye datos técnicos, descripción, confidence y warnings.

🔐 JWT

Registro, login y roles USER / ADMIN.

🚗 Anuncios

CRUD de vehículos con marca, modelo, precio, km y estado.

📸 Cloudinary

Subida y borrado de imágenes en la nube.

❤️ Wishlist

Guarda vehículos favoritos.

📊 Admin

Gestión de usuarios, anuncios y mensajes.
